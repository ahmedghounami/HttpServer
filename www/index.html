Webserv Dashboard

A simple web server written in C++ that implements the HTTP/1.1 protocol.
The project is a great opportunity to learn about the inner workings of web servers
and how they handle requests and responses.

team members:

⚙️ Tech Stack:

  • C++
  • HTTP/1.1 with GET, POST, DELETE methods
  • CGI (PHP, Python)
  • Session management & cookies

🧰 Features:

  • File upload & download
  • File deletion
  • Directory listing
  • Custom error pages
  • Session management
  • CGI support (PHP, Python)

🖥️ PHP FORM:

A simple form to demonstrate CGI with PHP.

🐍 Python FORM:

A simple form to demonstrate CGI with Python.

📤 File Upload:

Upload files directly to the server through the web interface.

📤 File Upload (CGI):

Upload files using CGI scripts for enhanced functionality.

📤 File Upload (CGI Python):

Upload files using CGI scripts in Python for enhanced functionality.

🗑️ File Deletion:

Easily delete files from the server.

📂 Directory Listing:

View the contents of directories on the server.

🚫 Custom Error Pages:

Custom error pages for better user experience.
